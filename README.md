# query-builder

Composable SQL query builder with type safety

## Installation

```bash
pip install query-builder
```

## Quick Start

```python
from query_builder import Query
client = Query()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
